# Descripción
Sistema inteligente capaz de medir, registrar y analizar en tiempo real el consumo de agua en viviendas o instalaciones, controlado desde una aplicación móvil.

# Introducción
El proyecto HydroSense surge ante la creciente necesidad de hacer un uso eficiente del agua, un recurso cada vez más limitado debido al cambio climático, el crecimiento poblacional y la falta de sistemas modernos de monitoreo.

En la mayoría de hogares, el consumo de agua no se mide en tiempo real y las personas no tienen forma de saber cuánta agua gastan en actividades comunes. Esto dificulta la adopción de hábitos responsables y limita la capacidad de detectar fugas o consumos inusuales.

HydroSense busca ofrecer una solución de bajo costo que permita medir el uso del agua de manera precisa, accesible y automática, brindando información útil para reducir el desperdicio y promover la conciencia ambiental.
